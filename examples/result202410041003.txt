# SUMMARY
Visual Studio Code's September 2024 update (version 1.94) introduces enhanced features like improved file search, source control graph, and Python test coverage.

# IDEAS:
- Enhanced file search functionality in Explorer view.
- Source Control Graph now includes more filtering options.
- Python tests can be run with coverage for detailed results.
- Migration to ESM improves VS Code's startup speed.
- Users can specify which account to use for extensions.
- Copilot offers code completions and chat in Native REPL.
- Drag and drop files for better chat context integration.
- New features to assist in setting up test environments.
- Multiple GitHub accounts can now be logged in simultaneously.
- MSAL-based Microsoft Authentication improves security.
- TypeScript 5.6 includes improved detection of common programming mistakes.
- Notebook editor supports multi-cursor editing across cells.
- New settings for customizing Source Control Graph view.
- Improved accessibility features guide users through options.
- Experimental features include semantic search and improved debugging commands.
- The command center offers quick access to chat commands.
- The ability to update extensions across multiple profiles.
- Inlay hint improvements enhance coding efficiency.
- JavaScript debugger now better displays HTML elements.
- Copilot can reference public code in chat responses.
